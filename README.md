# Week4_AnalystLab_Africa_Task
An interactive Power BI dashboard built with the Sample Superstore Sales dataset to analyse sales, profitability, customer behaviour, and product performance. Completed as part of Week 4 of the AnalystLab Africa Data Analytics Internship.

---

## Project Objectives

The dashboard answers key business questions such as:

* Which customer segment generates the most orders?
* Which products generate the highest revenue?
* Which customers contribute the most profit?
* How has revenue changed over time?
* Which shipping mode is most frequently used?
* Which states generate the most profit?

## Key Insights

* Consumer customers placed the highest number of orders (**3K**), followed by Corporate (**2K**) and Home Office (**1K**).
* Standard Class generated the highest sales (**$554K**) among all shipping modes.
* Chairs (**$1.7M**) and Phones (**$1.6M**) were the highest revenue-generating product sub-categories.
* Revenue grew steadily from **2015** to **2017**, reaching approximately **$3.6M** in 2017.
* Tamara Chand generated the highest customer profit (**$9K**), followed by Raymond Buch (**$7K**).

---

## Recommendations

* Increase investment in high-performing products such as Chairs and Phones.
* Reward and retain high-value customers through loyalty programs.
* Investigate low-profit states to improve profitability.
* Continue optimizing the Standard Class shipping process.
* Use historical sales trends to improve demand forecasting.
